# BLUE Data Layout

The public `data/` directory is intentionally minimal and only documents assets
that are already released in this repository.

## Current contents

- `routes/bench2drive_split/`: the 220 Bench2Drive route XML files used for
  Stage 1 evaluation.

## Release plan

- Stage 1: evaluation code, model checkpoints, evaluation logs, and route files.
- Stage 2: training data and training code.

Placeholder folders for unreleased data artifacts have been removed so that the
public tree matches the actual release scope.
