# BLUE Dataset

## Current repository contents

The current public repository only includes the route assets required for BLUE
evaluation:

- `data/routes/bench2drive_split/`

Stage 1 evaluation does not require the BLUE training dataset. Besides the route
XML files, it only depends on the official SimLingo backbone checkpoint.

## Upcoming Stage 2 release

Stage 2 will release the BLUE training data together with the corresponding
training code.

This repository no longer describes separate future placeholder releases beyond
that Stage 2 scope.

## Documentation policy

Until Stage 2 is published, the `data/` directory documents only assets that are
already present in this repository. Placeholder directories for unreleased
datasets have been removed to keep the public layout aligned with the current
release scope.
